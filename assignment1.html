Naser Alqseer's Assignment 1!

For my first, cookie cutter I used an MF DOOM mask reference image and adjusted it in photoshop and illustrator to be able to import it into rhino. Then, got the outlines of the image in rhino and offset it to be able to extrude the cookie cutter with reasonable thinkness of 0.2-0.3 in. lastly, created a support layer to be able to get the detail of the eyes. This process with repeated for the other reference images which are Aang and a D20.
